# Recipe App Backend
Routes: \
/get          -to get every Recipe \
/create       -to create a Recipe \
/getbyid      -to get a Recipe by id \
/select/:id   -to select a Recipe \
/deselect/:id -to deselect a Recipe \
/colormode    -to get/set the colormode(darkmode|lightmode) \
